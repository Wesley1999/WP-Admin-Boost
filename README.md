## 插件功能

使用jsdelivr加速WordPress的后台核心小文件与插件小文件，大幅提高后台访问速度。支持自定义加速插件。

## 注意事项

**请勿使用测试版**插件，请仔细检查加速后是否兼容。可以通过浏览器的Console报错检查兼容性。

此插件与jsdelivr无关，jsdelivr造成的意外损失与本插件无关。
